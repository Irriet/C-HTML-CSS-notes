Arrays

What are arrays?

Arrays are data structures that can store multiple variables of the same type. To declare an array you need to specify the type of its elements. Arrays can be single-dimensional or multidimensional. The size of an array is defined upon its creation by either stating the number or adding some elements.

How to create an array? Index of elements?

Below are some examples of array creating. It is crucial to remember that index numbers of the elements start from 0.

string[] pets = { "Rubin", "Lapis", "Onyx", "Oreo", "Kropek", "Piszczek"};
Console.WriteLine($"The array length is: {pets.Length}. The elements are: ");
foreach (string name in pets)
{
 Console.WriteLine(name);
}

In the example below, the 4 in the square brackets indicates the size of the array. Only 3 elements are added though, the last one remains empty. Upon displaying, the value for an empty int element is by default 0.
int[] numbers = new int[4];
numbers[0] = 1;
numbers[1] = 2;
numbers[2] = 3;

foreach (int number in numbers)
{
 Console.WriteLine($"Number: {number}.");
}

Number: 1.
Number: 2.
Number: 3.
Number: 0.

Is there a way to remove null elements?

Not really. Array.Resize() doesn't do that, there's no other method to really do that. The best way of achieving this goal is counting the elements that have values different than null by iterating through the elements and incrementing the counter - a variable. Then you can create a second array of the counter's size and loop the elements from the original array to copy the values different than null ones to the new array.

Connecting array elements back together (.Split(), .ToCharArray())
