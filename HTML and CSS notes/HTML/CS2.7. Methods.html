About methods

Method in C# is a separate code block that contains a series of statements to perform particular operations. Methods must be declared either in class or struct by specifying the required parameters. Generally they make the code more reusable - you can use the same function over and over without writting it multiple times - less code duplication.

Example of a method:
static void Main(string[] args)
{
 Console.WriteLine(GetPow(6, 3));
}

static int GetPow(int baseNum, int powNum)
{
 int result = 1;

 for (int i = 0; i < powNum; i++)
 {
  result = result * baseNum;
 }
 return result;
}

Static (not public) means that the method belongs to the (Program) class itself and not to its object (object methods are defined inside a class - more on that below). You don't need to create an object to access the method (no "new" keyword is used). For example Console.WriteLine(Math.Sqrt(144)); works just like that, no need for Math myMath = new Math();. The int informs us that the returned value is of integer type. A method that doesn't return a value would be void.
Example of a void method is presented below:

class UsefulTools
{
public static void SayHi(string name)
 {
  Console.WriteLine("Hello " + name);
 }
}


class Program
{
static void Main(string[] args)
 {
  UsefulTools.SayHi("Peter");
 }
}

Hello Peter

Note the UsefulTools. part - it marks the fact that the method is in different class and we need to point towards the class to access its method.
