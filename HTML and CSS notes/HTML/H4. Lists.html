Lists

Unordered lists

<ul>
<li>Fruits
 <ul>
  <li>Apples</li>
  <li>Grapes</li>
  <li>Oranges</li>
 </ul>
</li>
<li>Vegetables
 <ul>
  <li>Carrots</li>
  <li>Salads</li>
 </ul>
</li>
</ul>

Important!

Note the bolded </li> of Fruits and Vegetables. It should always be positioned after the sub-points of the main point. It's especially important for ordered lists, where putting the </li> right after the main point will ruin the numbering.

  • Vegetables

Ordered lists

You can use HTML tags to change the type of markers, like <ol type="1">, <ol type="I">, <ol type="A"> etc. It can be useful for creating sub-points, but generally it's a good practice to do it within the stylesheet.
<ol>
<li>Fruits
 <ol>
  <li>Apples</li>
  <li>Grapes</li>
  <li>Oranges</li>
 </ol>
</li>
<li>Vegetables
 <ol>
  <li>Carrots</li>
  <li>Salads</li>
 </ol>
</li>
</ol>

  1. Fruits
    1. Apples
    2. Grapes
    3. Oranges
  2. Vegetables
    1. Carrots
    2. Salads
