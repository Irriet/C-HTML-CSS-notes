Basic maths

Brackets are important

The programming language doesn't know what you mean unless you're very specific.
Example of simple math gone wrong is presented below (the third line) with a solution of the problem (the fourth line):

int widgetsSold = 6;
string firstName = "Tom";
Console.WriteLine(firstName + " sold " + widgetsSold + 7 + " widgets.");
Console.WriteLine(firstName + " sold " + (widgetsSold + 7) + " widgets.");

Tom sold 67 widgets.
Tom sold 13 widgets.
