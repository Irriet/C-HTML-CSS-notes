Basic HTML document structure

Head

Head is like settings: it contains all the data about the page.
In the head you can define the title, metadata, you can import any resources that are needed: fonts, stylesheets, etc.
Below you can find an example of HTML document's head:

<head>
<link href="https://fonts.googleapis.com/css?family=Lato:400,700|Sarabun:400,700" rel="stylesheet">
<link rel="stylesheet" type="text/css" href="../HTML/Stylesheet1.css" />
<meta charset="utf-8" />
<title>This is a website.</title>
</head>
