Classes and objects

Class is a data type that you can define by yourself.
Object is an instance of a class. It's an actual physical thing inside of a program. For example:

class Book
{
 public string title;
 public string author;
 public int pages;
}
static void Main(string[] args)
{
 {
  Book book1 = new Book();
  book1.title = "Harry Potter";
  book1.author = "J. K. Rowling";
  book1.pages = 400;
 }
}

Constructors

Constructors are special methods, which can be put inside of a C# class. A constructor is going to be called whenever you create an object of that class.

class Book
{
 public string title;
 public string author;
 public int pages;

 public Book(string aTitle, string aAuthor, int aPages) //this is a constructor
 {
  title = aTitle;
  author = aAuthor;
  pages = aPages;
 }
}

class Program
{
 static void Main(string[] args)
 {
  {
   Book book1 = new Book("Harry Potter", "J. K. Rowling", 400);
   Book book2 = new Book("Lord of the Rings", "Tolkien", 700);
   Console.WriteLine(book2.title);
  }
 }
}
