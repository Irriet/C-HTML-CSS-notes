Google's Homepage

Google's Homepage

Images

Setting width&height, alt attribute

You will notice width attribute in the code boxes of this section. These values are pixels. You may add both width and height, but it's good to keep the ratio, so the pictures don't get smushed. If you set only one value (either width or height), the other one will scale automatically.

The alt attribute is a text that will appear when an image cannot be found.

Adding images

To add an image from own pc:
<img src="media/Small Onyx.jpg" img width="300" alt="A really cute rat">


To link a picture from the Internet:
<img src="https://i.pinimg.com/originals/f7/40/17/f7401734665945e1a573ec5e7ca0edba.jpg" width="300" alt="A really cute dumbo rat">


To use an image as a link:
<a href="https://www.google.com target="_blank""><img src="media/Small Lapis.jpg" img width ="300" alt="A really cute rat"></a>


Videos

To add a video from own pc:
<video src="" poster="ThumbnailImage.jpg" controls width="500">Here we put the text that will display if video fails to play.</video>

The controls creates browser default controls, so it's really important to add it.

To add a video from YouTube you go to the video, click Share → Embed and copy the code from there. Example of a copied code:
<iframe width="560" height="315" src="https://www.youtube.com/embed/lw_95WzLk7U" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
