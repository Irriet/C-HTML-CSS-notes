Containers

Containers are used to wrap other HTML tags and they consist of a starting and ending tag. They are used mostly to keep the code readable and to add styling to the document. To explain two types of containers, you need to know the two main ways of displaying the content in HTML: inline and block.

Inline

Inline elements only take as much width on the screen as they need to, like links. A container for inline styling is called span.
