The difference between loops

  • foreach - executes once for each element in the data sequence, like an array. With foreach you can't reassign a variable's value.
  • for - executes the code a certain number of times. For enables changing the value of the variable and iterating through an array backwards.
  • while - executes till a certain condition is met
  • do-while - very similar to while with one difference: it is always executed once, before the conditions are verified

Foreach

string[] petNames = { "Ruby", "Lapis", "Onyx" };
foreach (string name in petNames)
{
Console.WriteLine(name);
}

Instead of variable type name (here string name) you can use var keyword to let the compiler decide what is it.

For

Consists of the following elements:
for (initiator; condition; iterator)

Initializer is a variable that is initialized. Condition defines the loop completion condition. Iterator defines the action that is to be taken after each iteration. And then in the { } is the body - a block of code that will be executed for each iteration.

Below you can find the simplest example of the for loop. It includes the break keyword. You can read more about this a bit below in the last sub-section.

for (int i = 0; i < 10; i++)
{
Console.WriteLine(i);
if (i == 7) break;
}

for (int i = petNames.Length - 1; i >= 0; i--)

string[] petNames = { "Ruby", "Lapis", "Onyx", "Piszczek", "Kropek", "Snickers" };

Break & Continue

Break

break can be used in a switch statement, like shown in the Logic section. It also functions in all sorts of loops, as a statement used to jump out of a loop. In the code below, the loop will end once i is equal 4.
for (int i = 0; i < 7; i++)
{
 if (i == 4)
 {
  break;
 }
 Console.WriteLine(i);
}

It's worth noting that depending on where you put the break statement the output will or will not contain the number 4. If you put break after Console.WriteLine(i) the number 4 will still be printed out.

Continue

The continue statement breaks one iteration in the loop upon meeting a certain condition, but continues with the next iteration.
for (int i = 0; i < 7; i++)
{
 if (i == 4)  {
  continue;
 }
 Console.WriteLine(i);
}
