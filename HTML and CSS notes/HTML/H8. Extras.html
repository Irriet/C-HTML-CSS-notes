Metatags

Metatags are in the head of the HTML document.


<head>
<meta charset="UTF-8">
<meta name="description" content="A website for C#, HTML, CSS notes">
<meta name="author" content="Madzia">
<meta name="keywords" content="HTML, C#, CSS, tutorials, self teaching">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

Description is meant to always be short. The keywords don't matter that much anymore, but you can put them there if you want. Viewport controls how the website is displayed on different devices.

iFrame

iFrame is a feature used to embed an entirely different website inside of your website. Well-known pages won't allow this.


<iframe src="https://www.mikedane.com/" frameborder="0" width="900" height="500">This is Giraffe Academy.</iframe>
