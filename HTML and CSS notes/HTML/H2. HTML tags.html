HTML tags

How does it work?

There are various types of tags in HTML. Tags can be used for containing text (in order to style it easier later on), for adding things like links, videos, pictures, for making lists, defining headers in the text and so on. There is also a special <meta> tag that allows you to define the metadata (information about data) of your document: charset, keywords, author and others. Some additional information about metatags can be found in the "Extras" section.

Some examples of HTML tags are:

  • <h1> - <h6> for headers
  • <b> <i> <u> for bold, italic, underline - although it's better to define such things in stylesheets
  • <p> for new paragraph
  • <br /> for an empty line
  • <hr> for a horizontal separating line
  • <header> <main> <footer> for dividing the document into sections
  • <nav> for creating navigation, mainly in header
  • <article> for creating blog posts, etc; mainly in <main>
