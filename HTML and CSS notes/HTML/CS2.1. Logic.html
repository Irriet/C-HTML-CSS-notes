Operators

For writing any logic code it's important to know the basic operators, so the program understands whether you want the values to be equal, not equal, greater or less than another one and so on. Below some basic operators are presented.

Basic logic operators
Operator What it means Code example or notes
== equality A
!= inequality
!bool bool value false B
> greater than
< less than
>= greater than or equal
<= less than or equal
&& and for connecting conditions
|| or if one or another condition is true

Code examples

A:

Console.WriteLine("a" == "a");
Console.WriteLine("a" == "A");
Console.WriteLine(1 == 2);

string value = "a";
Console.WriteLine(value == "a");

true
false

string sentence = "Oreo is a cat.";

Console.WriteLine(!sentence.Contains("rat"));
