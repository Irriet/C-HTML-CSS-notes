Operators

For writing any logic code it's important to know the basic operators, so the program understands whether you want the values to be equal, not equal, greater or less than another one and so on. Below some basic operators are presented.

If

if (this condition is true)
 {
  Execute this code block
 }
else if (this condition is true)
 {
  Execute this code block
 }
else
 {
  If none of the above conditions returned true, execute this code block
 }

Conditional operator ?:;

This works a little bit like a simplified "if" statement. The basic idea is presented below:
<evaluate this condition> ? <if condition is true, return this value> : <if condition is false, return this value>

Example:
int totalPrice = 1001;
int discount = totalPrice > 1000 ? 100 : 50;
Console.WriteLine($"The discount is {discount}$.");

The discount is 100$.

Switch

int number = 3;
string letter = "";

switch (number)
{
 case 1:
  letter = "a";
  break;
 case 2:
  letter = "b";
  break;
 case 3:
  letter = "c";
  break;
 case 4:
  letter = "d";
  break;
}
Console.WriteLine($"The letter corresponding to the number is: {letter}.");

The letter corresponding to the number is: c.
