Loop Challenges

Sum of array elements

You can do that using any sort of loop. The results of codes presented below are all the same.

With Foreach

decimal[] prices = { 12.50m, 19.84m, 51.63m, 98.98m, 32.45m, 57.00m };
decimal sum = 0;
int counter = 0;

foreach (decimal num in prices)
{
 sum += num;
 counter++;
 Console.WriteLine($"Item nr {counter}: {num}$, sum: {sum}$");
}
Console.WriteLine($"Total is: {sum}$.");

Item nr 1: 12,50$, sum: 12,50$
Item nr 2: 19,84$, sum: 32,34$
Item nr 3: 51,63$, sum: 83,97$
Item nr 4: 98,98$, sum: 182,95$
Item nr 5: 32,45$, sum: 215,40$
Item nr 6: 57,00$, sum: 272,40$
Total is: 272,40$.

With For

decimal[] prices = { 12.50m, 19.84m, 51.63m, 98.98m, 32.45m, 57.00m };
decimal sum = 0;
int counter = 0;

for (int i = 0; i < prices.Length; i++)
{
 sum += prices[i];
 counter++;
 Console.WriteLine($"Item nr {counter}: {prices[i]}$, sum: { sum}$");
}
Console.WriteLine($"Total is: {sum}$.");

While

decimal[] prices = { 12.50m, 19.84m, 51.63m, 98.98m, 32.45m, 57.00m };
decimal sum = 0;
int counter = 0;

while (counter < prices.Length)
{
sum += prices[counter];
Console.WriteLine($"Item nr {counter + 1}: {prices[counter]}$, sum: { sum}$");
counter++;
}
Console.WriteLine($"Total is: {sum}$.");

Counting how many times a certain element appeared in an array

int[] numbers = { 1, 2, 3, 4, 1, 5, 1, 8, 4, 4, 2, 1, 7, 1 };
int counter = 0;
foreach (int num in numbers)
{
 if (num == 1)
  counter++;
 else
  continue;
}

The number has appeared 5 times.

Finding the highest number in an array

int[] numbers = { 3, 56, 74, 35, 87, 101, 34, 65, 5, 8, 18, 93 };
int highNum = 0;

foreach (int number in numbers)
{
 if (number > highNum)
 {
  highNum = number;
 }
 else
  continue;
}
Console.WriteLine($"The highest number is: {highNum}.");

The highest number is: 101.

Finding the lowest number in an array

int[] numbers = { 3, 56, -64, 74, 35, -10, 87, 101, 34, 65, 5, 8, 18, 93 };
int lowNum = numbers[0];

foreach (int num in numbers)
{
 if (num < lowNum)
  lowNum = num;
 else
  continue;
}
Console.WriteLine($"The lowest number is: {lowNum}");

The lowest number is: -64
