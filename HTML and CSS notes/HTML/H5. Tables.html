Tables

The basic tags showed in the example below create a table without any borders.
To add a border, simply add value for the border attribute in your stylesheet, like this one:
table, th, td {
border: 1px solid rgb(34, 34, 59);
}

Table code:
<table>
 <caption>
  <h3>Basic logic operators</h3>
 </caption>
 <thead>
  <tr>
   <th>Operator</th>
   <th>What it means</th>
   <th>Code example or notes</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>==</td>
   <td>equality</td>
   <td>A</td>
  </tr>
  <tr>
   <td>!=</td>
   <td>inequality</td>
   <td></td>
  </tr>
  <tr>
   <td>!bool</td>
   <td>bool value false</td>
   <td>B</td>
  </tr>
 </tbody>
</table>

Basic logic operators
Operator What it means Code example or notes
== equality A
!= inequality
!bool bool value false B
> greater than
< less than
>= greater than or equal
<= less than or equal
&& and for connecting conditions
|| or if one or another condition is true
